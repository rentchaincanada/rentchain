# Governance

This folder contains the policy, approval, audit, and runbook material used to govern agent-assisted development in RentChain.

## Structure
- agent-policies/: policy presets with scoped permissions and limits
- approval-rules/: review expectations and protected-area guidance
- audit/: audit trail documentation and run-report expectations
- runbooks/: operator procedures for standard runs, emergency stop, and rollback
- task-templates/: mission templates for common task types
